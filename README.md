# VandaCoerulea

Android aplikacija razvijena u **Unity** engineu kao projekat za predmet **Računarska grafika**. Aplikacija prikazuje interaktivnu 3D scenu orhideje *Vanda coerulea* s podrškom za proširenu stvarnost (AR).

---

## Funkcionalnosti

- **3D model orhideje** — detaljan model biljke *Vanda coerulea* s primijenjenim teksturama dizajniran u Blenderu
- **Interaktivna scena** — pregled 3D modela i interakcija sa tačkama interesa na sceni
- **AR (proširena stvarnost)** — prikaz modela u stvarnom prostoru putem kamere uređaja
- **Animacije** — animirani elementi scene

---

## Grafičke tehnike

- **Teksture** — mapiranje tekstura na 3D model
- **Osvjetljenje (Lighting)** — realistično osvjetljenje scene
- **Sjene (Shadows)** — dinamičke sjene objekata

---

## Tehnologije

- **Unity** — game engine za razvoj 3D/AR aplikacija
- **Blender** - razvoj realističnog 3D modela orhideje
- **Android** — ciljna platforma

---

## Instalacija

APK fajl je dostupan u [Releases](https://github.com/aminamemiic/VandaCoerulea/releases).

### Koraci za instalaciju

1. Preuzmi `VandaCoerulea.apk` sa [Releases](https://github.com/aminamemiic/VandaCoerulea/releases) stranice
2. Na Android uređaju omogući instalaciju iz nepoznatih izvora:
   - **Postavke → Sigurnost → Nepoznati izvori** ✓
3. Otvori preuzeti APK fajl i klikni **Instaliraj**
4. Pokreni aplikaciju

> ⚠️ Aplikacija zahtijeva Android uređaj.

---

## 🛠️ Pokretanje iz izvornog koda

### Preduvjeti

- [Unity](https://unity.com/)
- Android Build Support modul
- Android SDK i JDK

### Koraci

1. Kloniraj repozitorij:
   ```bash
   git clone https://github.com/aminamemiic/VandaCoerulea.git
   ```
2. Otvori projekt u Unity editoru
3. U **Build Settings** odaberi **Android** kao ciljnu platformu
4. Klikni **Build & Run** ili **Build** za generisanje APK fajla
